代理工具 （proxychain）

## 安装 proxychains
sudo apt-get install proxychains
报错：ERROR: ld.so: object ‘libproxychains.so.3’ from LD_PRELOAD cannot be preloaded (cannot open shared object file): ignored.

解决方法：

## 搜索 libproxychains.so.3 文件
➜  ~  find /usr/ -name libproxychains.so.3 -print
/usr/lib/x86_64-linux-gnu/libproxychains.so.3

## 编辑 /usr/bin/proxychains 文件
export LD_PRELOAD=libproxychains.so.3
改为
export LD_PRELOAD=/usr/lib/x86_64-linux-gnu/libproxychains.so.3

QQ 无法加载图片

  1. 禁用 ipv6 - 方案 1
# 1、修改系统信息配置文件
sudo vim /etc/sysctl.conf

# 2、在文件中添加如下内容
# IPv6 disabled
net.ipv6.conf.all.disable_ipv6 =1
net.ipv6.conf.default.disable_ipv6 =1
net.ipv6.conf.lo.disable_ipv6 =1

# 3、重新加载配置文件
sudo sysctl -p
  1. 禁用 ipv6-方案 2
# 1、修改 gurb 文件
sudo  vim /etc/default/grub

# 2、添加如下内容
GRUB_CMDLINE_LINUX="ipv6.disable=1"

# 3、重新生成 gurb
sudo update-grub2

secureCRT 运行报错：SecureCRT: error while loading shared libraries: libpng12.so.0: cannot open shared object file: No such file or directory

解决方法：

sudo apt install libpng12-0

安装 sudo apt install libpng12-0 报错：无法安装 /lib/x86_64-linux-gnu/libpng12.so.0 的新版本: 没有那个文件或目录

解决方法：

  • 下载文件
    下载 libpng12-0

  • 复制文件

sudo cp libpng12.so.0 /lib/x86_64-linux-gnu

问题原因：
因为 /lib/x86_64-linux-gnu 目录作用的 libpng 是 16 版本，所以低版本的 libpng12.so.0 无法自动安装到目录中

桌面美化

grub 主题美化

下载 grub 主题

https://www.gnome-look.org/browse/cat/109/ord/rating/

创建 grub 主题文件夹

sudo mkdir /boot/grub/themes

解压缩主题文件

sudo xz -d *****.tar.xz
sudo tar -xvf *****.tar -C /boot/grub/themes

修改配置文件

# 查找并复制 theme.txt 文件路径
find /boot/grub/themes -name theme.txt

# 修改 00_header 文件
sudo vim /etc/grub.d/00_header
# 添加如下内容
GRUB_THEME="/boot/grub/themes/***/****/theme.txt"
GRUB_GFXMODE="1920x1080x32"

更新配置

sudo update-grub
